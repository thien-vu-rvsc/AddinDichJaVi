JP-VI Translator Add-in

Hướng dẫn khởi chạy ứng dụng trực tiếp trên Microsoft Office (Windows)

Tổng quan về phiên bản Serverless

Phiên bản Add-in này hoạt động hoàn toàn không cần backend server chạy cục bộ (loại bỏ Python FastAPI và các mô hình LLM). Add-in sẽ kết nối trực tiếp đến API Google dịch từ trình duyệt và tự động sinh phiên âm tiếng Nhật (Hiragana/Romaji) thông qua bộ xử lý client-side cực nhẹ.

Ưu điểm: Khởi chạy tức thì, cực kỳ nhẹ máy, không chiếm dụng tài nguyên CPU/GPU và không cần cài đặt Python. Chỉ cần máy tính của bạn có kết nối Internet để gọi API Google dịch.

Các bước khởi chạy Add-in

1

Yêu cầu chuẩn bị

Đảm bảo máy tính của bạn đã cài đặt Node.js (Phiên bản khuyến nghị: v18 trở lên). Nếu chưa có, bạn có thể tải về từ Trang chủ Node.js.

2

Mở Terminal tại thư mục dự án

Mở cửa sổ Command Prompt (cmd) hoặc PowerShell tại thư mục chứa mã nguồn Add-in này.

3

Cài đặt các gói thư viện phụ thuộc (Chỉ chạy lần đầu)

Nhập lệnh sau và nhấn Enter để cài đặt thư viện:

npm install
4

Khởi chạy Add-in lên ứng dụng Office mong muốn

Hệ thống hỗ trợ khởi chạy tự động lên **Excel, Word hoặc PowerPoint**. Hãy chạy một trong các lệnh dưới đây tương ứng với ứng dụng bạn muốn sử dụng:

Dành cho Microsoft Excel:

npm run start:excel

Dành cho Microsoft Word:

npm run start:word

Dành cho Microsoft PowerPoint:

npm run start:powerpoint
Lần chạy đầu tiên: Một hộp thoại Windows sẽ xuất hiện yêu cầu bạn cấp quyền tin cậy Chứng chỉ SSL cục bộ (Developer Certificate) để đảm bảo kết nối HTTPS an toàn. Vui lòng bấm Yes / Đồng ý để tiếp tục.

Cách kích hoạt giao diện Add-in

Sau khi chạy lệnh, Microsoft Office sẽ tự động mở ra. Bạn hãy thao tác như sau để kích hoạt bảng điều khiển:

1. Trên thanh công cụ, vào thẻ Home

Tìm mục Add-ins (hoặc My Add-ins) ở góc bên phải.

2. Chọn Add-in phát triển cục bộ

Chọn mục **Developer Add-ins** (hoặc *Add-in dành cho nhà phát triển*), biểu tượng JP-VI Translator sẽ xuất hiện tại đó. Click vào để mở thanh dịch thuật ở góc phải.

Microsoft Excel

Tự động dịch dữ liệu của ô bảng tính khi click chọn.

Microsoft Word

Dịch tức thì đoạn văn bản bôi đen trong tài liệu.

PowerPoint

Dịch nội dung văn bản trên slide đang biên tập.

Xử lý sự cố thường gặp

Lỗi chặn chứng chỉ bảo mật (SSL Certificate)

Nếu Office hiển thị lỗi "ADD-IN ERROR: The content is blocked because it isn't signed by a valid security certificate", có nghĩa là Office chưa tin cậy chứng chỉ SSL local.

Cách khắc phục: Chạy lệnh sau trong Terminal để cài đặt lại chứng chỉ SSL:

npx office-addin-dev-certs install

Sau đó đồng ý ở thông báo Windows xuất hiện và khởi chạy lại lệnh ở Bước 4.

Dừng ứng dụng

Khi sử dụng xong, bạn có thể nhấn Ctrl + C tại cửa sổ terminal để tắt server. Nếu muốn gỡ Add-in ra khỏi Office, hãy chạy lệnh:

npm run stop
