Yêu cầu thiết bị hỗ trợ **WebGPU** (Microsoft Edge trên Win 10/11 có GPU rời hoặc GPU tích hợp). Trọng số mô hình sau khi tải thành công sẽ được lưu trữ trong trình duyệt (Cache Storage) và hoạt động hoàn toàn offline cho các lần sau.
Chưa kết nối
Dành cho máy tính không có GPU, chỉ có CPU. Vui lòng cài đặt Ollama và chạy lệnh sau trong cmd/terminal của máy bạn để tải mô hình trước khi dịch: ollama run gemma4:e2b
Chưa kết nối
Dịch thông qua **Antigravity SDK Agent** chạy trên máy chủ Python cục bộ (FastAPI). Hãy đảm bảo bạn đã chạy server backend tại cổng 8000 và đã cài đặt thư viện: pip install fastapi uvicorn google-antigravity
